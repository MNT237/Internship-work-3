CLUSTERS


🤖 sklearn.cluster

Part of Scikit-learn, provides popular clustering algorithms

Includes KMeans, DBSCAN, AgglomerativeClustering & more

Groups unlabeled data into clusters based on similarity — no target variable needed

KMeans is most common: specify n_clusters, fit your data, done


🎨 Seaborn

Built on top of Matplotlib — produces prettier, statistical plots with less code

Best for: heatmaps, pairplots, boxplots, violin plots & countplots

Natively works with pandas DataFrames — just pass df and column names

Ideal for visualizing correlations, distributions, and categorical comparisons
